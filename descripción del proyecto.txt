1. Objetivo general del CRM
Desarrollar un CRM de ventas integrado mediante API que permita gestionar socios comerciales, clientes, contactos, tareas de ventas, seguimiento de presupuestos y actividades de ventas desde una interfaz React, utilizando un backend en Python y conectándose a la misma base de datos mediante API.

2. Concepto general
El CRM debe funcionar como una capa de ventas y relaciones sobre la información existente en otro sistema de gestión. El sistema debe permitir centralizar la información de ventas, los contactos, el seguimiento y las tareas relacionadas con los clientes y las operaciones.

3. Gestión de socios comerciales

14. Recomendaciones técnicas
• Utilizar API para la integración
• Separar la lógica del CRM de la lógica operativa
• Los permisos de usuario se centralizan en el software de gestión y se comunican mediante API.

• Diseñar componentes reutilizables en React
• Preparar una estructura multiusuario
• Preparar la integración futura con IA y automatización
